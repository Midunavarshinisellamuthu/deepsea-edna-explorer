Upload Guidelines

For optimal analysis results, please follow these guidelines:

  • Use FASTA or FASTQ format for sequence files
  • Ensure sequences are properly formatted
  • Include metadata when available for richer analysis
  • Provide accurate collection information
  • Larger files (>100MB) may take longer to upload
Analysis Options

The DeepSea eDNA Explorer pipeline includes:

  1. Preprocessing: Quality filtering, adapter trimming, and length filtering
  2. Classification: Deep learning-based taxonomic classification
  3. Annotation: Detailed taxonomic annotation
  4. Abundance Estimation: Relative abundance calculation
  5. Visualization: Interactive charts and reports
